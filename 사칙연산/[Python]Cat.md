# [Python] 고양이

### [문제]

고양이를 출력하라...(이게 고양이냐?)(똥손)(집사가 미안해)

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

### [코드]

```python
print('\    /\ \n )  ( \') \n(  /  ) \n \(__)|')
```

### [해설]

- 줄바꿈 문자인 \n를 사용하여 줄바꿈을 해주고 다음글 출력
- 출력시 문제가 될 수 있는, 구분 문자들이 있는경우 ( ' , [ ] ( )  등등)  \  슬래시를 앞에넣어주면 string으로 인식되어 출력됨
