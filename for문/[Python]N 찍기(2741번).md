# [Python] N 찍기 (2741번)

### [문제]

자연수 N이 주어졌을 때, 1부터 N까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

### [입력/출력]

첫째 줄에 100,000보다 작거나 같은 자연수 N이 주어진다.

```
첫째 줄부터 N번째 줄 까지 차례대로 출력한다.
입력: 5
출력:
1
2
3
4
5
```

### [코드]

```python
for i in range(1, 1+int(input())): print(i)
```

### [해설]

- for문에 range() 함수로 1부터 입력받은 값보다 1만큼 큰 숫자의 범위만큼
- 순서대로 출력
